# Vibha Events

Architecting high-fidelity experiences for those who define the future.

Vibha Events is a luxury event planning application designed with a high-end, architectural aesthetic. It provides a bespoke interface for configuring elite event blueprints, showcasing past masterpieces, and connecting with a global concierge team.

## Features

- **Architectural UI**: A sophisticated dark-themed interface with gold accents and high-fidelity typography.
- **Engine Room**: A specialized event configuration tool for selecting event types, guest counts, and required services.
- **Masterpieces Gallery**: A curated, filterable showcase of past events with interactive hover states.
- **Bespoke Consultation**: Integrated consultation request system with real-time feedback.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop experiences.
- **Animations**: Smooth, high-end transitions powered by `motion/react`.

## Tech Stack

- **Frontend**: React 18+, TypeScript, Vite
- **Styling**: Tailwind CSS
- **Animations**: Motion (formerly Framer Motion)
- **Icons**: Lucide React
- **Notifications**: Sonner

## Getting Started

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
